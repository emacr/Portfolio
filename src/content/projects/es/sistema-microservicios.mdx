## Contexto del Proyecto

Este proyecto resolvió la sobrecarga operativa de un restaurante de comida rápida que recibía pedidos por WhatsApp de forma manual, lo que generaba errores y tiempos de respuesta lentos. La automatización con n8n permitió gestionar todos los pedidos de manera ordenada y eficiente.

## Flujo de Automatización

El cliente envía su pedido por WhatsApp → Evolution API captura el mensaje → n8n procesa y valida el pedido con IA (ChatGPT/DeepSeek) → el pedido se registra en PostgreSQL → Chatwoot notifica al equipo del restaurante y confirma automáticamente al cliente.
