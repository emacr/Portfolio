## Contexto del Proyecto

Este proyecto nació de la necesidad de practicar la integración Fullstack entre un ecosistema moderno de Frontend como React y una arquitectura robusta de Backend utilizando Java. La aplicación permite a los usuarios registrarse y gestionar sus pensamientos de forma privada y segura.
