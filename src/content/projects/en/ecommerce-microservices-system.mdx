## Project Context

This project solved the operational overload of a fast-food restaurant that was manually handling WhatsApp orders, causing errors and slow response times. The n8n automation allowed all orders to be managed in an orderly and efficient way.

## Automation Flow

The customer sends their order via WhatsApp → Evolution API captures the message → n8n processes and validates the order with AI (ChatGPT/DeepSeek) → the order is registered in PostgreSQL → Chatwoot notifies the restaurant team and automatically confirms to the customer.
