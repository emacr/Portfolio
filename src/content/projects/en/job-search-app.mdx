## Project Context

The objective of this application is to reduce friction in the search for a first job for young professionals. Through a friendly interface developed in .NET MAUI, candidates can filter positions based on their skills without prior experience.

## Technical Solution

The architecture is completely separated into two repositories. The Frontend offers a cross-platform native experience using Maui, while the robust .NET Backend handles the business logic and persistence in SQL Server.
